# testChatGpt

Mini juego de rol por turnos en consola donde el héroe gana experiencia, sube de nivel y mejora sus estadísticas al derrotar enemigos.

## Cómo jugar

```bash
python rpg_game.py
```

El juego muestra tus estadísticas y las del enemigo en cada turno. Puedes atacar o curarte, y al ganar combates obtienes experiencia para subir de nivel.
